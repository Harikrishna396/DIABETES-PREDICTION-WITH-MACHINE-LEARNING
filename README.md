The project uses the **PIMA Indians Diabetes Dataset** for training and testing the machine learning model.

The dataset contains **768 medical records** with **8 input features** and **1 target output column**.

### Dataset Features

| Feature | Description |
|---------|-------------|
| Pregnancies | Number of times the patient was pregnant |
| Glucose | Plasma glucose concentration |
| BloodPressure | Diastolic blood pressure measurement |
| SkinThickness | Triceps skin fold thickness |
| Insulin | Serum insulin level |
| BMI | Body Mass Index |
| DiabetesPedigreeFunction | Diabetes hereditary risk value |
| Age | Age of the patient |
| Outcome | Target variable (0 = Non-Diabetic, 1 = Diabetic) |

### Dataset Shape


Number of Samples: 768
Number of Input Features: 8
Target Variable: Outcome

Pregnancies

Outcome
